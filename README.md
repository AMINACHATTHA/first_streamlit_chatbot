# 🤖 First Streamlit Chatbot

A simple AI chatbot built using **Streamlit** and **Groq LLM**, deployed on **Hugging Face Spaces**.

---

## 🚀 Live Demo

👉 Try the chatbot here:


https://huggingface.co/spaces/aminachattha/first_streamlit_chatbot

---

## ✨ Features

* 💬 ChatGPT-like interface
* 🤖 Powered by Groq (Llama 3.1)
* ⚡ Fast AI responses
* 🧠 Session-based chat memory
* 🎨 Clean Streamlit UI
* ⌨️ Typing animation

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Groq API

---

## 📦 Installation (Local Setup)

```bash
git clone https://github.com/your-username/first_streamlit_chatbot.git
cd first_streamlit_chatbot
pip install -r requirements.txt
```

---

## 🔐 Environment Variables

Create a `.env` file:

```bash
GROQ_API_KEY=your_api_key_here
```

---

## ▶️ Run Locally

```bash
streamlit run app.py
```

---

## ☁️ Deployment

This project is deployed on **Hugging Face Spaces** using:

* Streamlit SDK
* Environment Secrets for API key

## 🧠 How It Works

1. User enters a message
2. Message is sent to Groq API
3. LLM processes input
4. Response is displayed in chat UI
